From the project root:

```bash

```

### Run a single test class (PowerShell/Windows)


### Parallel execution & max browsers

Parallelism is configured in `src/test/resources/junit-platform.properties`:

- methods and classes run concurrently
- fixed parallelism = `3`

Additionally, the framework enforces a **hard limit** of 3 concurrent browsers via a `Semaphore` in `SpringSeleniumTestBase`, so even if you change JUnit parallelism you wonвЂ™t exceed 3 browser instances.
